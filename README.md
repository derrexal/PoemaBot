# PoemaBotGitHub

Этот телеграмм бот позволяет легко и в удобной форме получать произведения русских поэтов.
Ежедневно в 7:00 он присылает по одному, наиболее подходящему вам стихотворению.
Кроме этого, бот позволяет получить стихотворение в любой удобный для вас момент времени.

В разработке функционал по фильтрации произведений. 
Например, можно будет получать стихотворения только от конкретных авторов, или только по определенным категориям ("любовь","жизнь","война и т.д.").

Для запуска проекта склонируйте данный репозиторий, после чего в файле /TgBotClient/Settings.cs измените поле "токен" на тот, который вам выдал BotFather.
